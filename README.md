BT-Search
=========
简洁舒服的BT搜索程序，居家旅游必备神器。


新版本开发中:
	* 使用Laravel开发，高效优雅
	* 对源站做新的适配

###运行环境
* PHP5.4+
* 开启CURL扩展
* 保留30MB以上的硬盘空间
(缓存会占用很多，100PV大概生成20M数据)

###安装说明
将所有文件上传到网站目录，新建cache目录并设置权限为777，并对config.php进行修改。
要求不高，大家给我保留个版权就可以了，链接到https://github.com/kslr/BT-Search
